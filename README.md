# react-native-culqi

## Getting started

`$ npm install https://github.com/czegarram/react-native-culqi --save`

## Usage
```javascript
import Culqi from 'react-native-culqi';

Culqi.createToken({
  card_number:'4111111111111111',
  cvv:'123',
  expiration_month:'09',
  expiration_year:'2020',
  email:'richard@piedpiper.com',
}, TOKEN_API)

```
